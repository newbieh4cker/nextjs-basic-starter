## 점검 항목

### 1. TypeScript 타입 체크
- `npx tsc --noEmit` 실행
- 타입 오류가 있으면 파일명과 오류 내용을 정리

### 2. ESLint 검사
- `npm run lint` 실행
- 린트 오류/경고가 있으면 파일명과 내용을 정리

### 3. 빌드 검증
- `npm run build` 실행
- 빌드 실패 시 오류 내용을 정리

### 4. 변경 파일 리뷰
- `git diff --cached` 및 `git diff`로 변경사항 확인
- 명백한 실수가 있는지 검토 (console.log 잔존, 하드코딩된 시크릿, 주석 처리된 코드 등)
